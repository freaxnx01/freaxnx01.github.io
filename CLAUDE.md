# CLAUDE.md

Personal site for `github.freaxnx01.ch` (GitHub Pages, served from `master`).

- Root landing page: `index.md` (Jekyll, `jekyll-theme-hacker`).
- **Browser Games Hub** at `/games/` — static `games/index.html` + `style.css` +
  `lightbox.js` + `assets/`. Screenshots are generated by
  `scripts/capture_screenshots.py`.

Each game lives in its own `game-<name>` repo (GitHub Pages, root `index.html`),
is served at `https://github.freaxnx01.ch/<repo>/`, and gets a card in
`games/index.html`.

## Game ideas / backlog

New game ideas and the concept backlog are kept in a **separate private repo**
(deliberately not named or linked here, to keep it out of this public repo). When
an idea matures it graduates to a `game-<name>` repo and gets a card on the hub.
